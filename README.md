# LoVR: A Benchmark for Long Video Retrieval in Multimodal Contexts

This is the repository that contains source code for the LoVR website.

## About LoVR

LoVR is a benchmark specifically designed for long video-text retrieval. It contains 467 long videos and over 40,804 fine-grained clips with high-quality captions.

## Dataset

The dataset is available on HuggingFace: [https://huggingface.co/datasets/debugger123/LoVR-benchmark](https://huggingface.co/datasets/debugger123/LoVR-benchmark)

## Code

The code repository is available at: [https://github.com/TechNomad-ds/LoVR-benchmark](https://github.com/TechNomad-ds/LoVR-benchmark)
